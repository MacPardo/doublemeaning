DOUBLE MEANING

BUT WHY?

Exitem muitos aplicativos/sites excelentes para aprender inglês, mas eles pecam em um detalhe muito importante: Eles não levam em consideração os múltiplos significados das palavras.

Tomemos o Anki como exemplo. Quando você acerta a resposta de um flashcard, ele registra que você memorizou aquela palavra, e você passa a ver ela com menos frequência. Porém, quando você aprende um novo significado para aquela palavra, o Anki não consegue identificar que ela deveria ser ensinada com este outro significado, tornando o sistema de repetição espaçada inútil para esta palavra.

O mesmo ocorre com outros softwares, como o LINQ, LWT, etc.

A proposta do Double Meaning é corrigir este problema. Por enquanto a única coisa que o Double Meaning faz é listar algumas palavras com múltiplos significados. A proposta é se tornar um web app que se comporta de forma parecida com o LINQ e o Anki, mas que trata cada significado de cada palavra de forma individual.
