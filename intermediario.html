DOUBLE MEANING
INICIANTE INTERMEDIÁRIO AVANÇADO

SET

  • Set pode passar a ideia de colocar algo em algum lugar
    • Can you set the plates on the table?
    • Você pode colocar os pratos na mesa?
  • Mas também pode signifcar conjunto
    • The set of all sets that are not members of themselves.
    • O conjunto de todos os conjuntos que não são membros de si mesmos.

MINE

  • O sentido mais empregado para mine é o de posse
    • This car is mine
    • Este carro é meu
  • Também pode querer dizer mina (de mineiração)
    • I don't like to work at the coal mine
    • Eu não gosto de trabalhar na mina de carvão
  • E ainda poder se referir a uma mina terrestre
    • Don't step on the mine, or it will explode!
    • Não pise na mina, ou ela vai explodir!

RUN

  • Run geralmente significa correr.
    • You need to run faster!
    • Você tem que correr mais rápido!
  • Tamém pode significar ser candidato à uma posição
    • You should run for president.
    • Você deveria se candidatar à presidência.
  • E ainda pode ser gerenciar
    • I am running a new business.
    • Eu estou gerenciando um novo negócio.
